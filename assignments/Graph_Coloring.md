과제14 - 그래프 - 칼러링

문제: 최소 그래프 컬러 수 구하기 (Graph Coloring – Chromatic Number)
학습 목표
그래프의 인접 리스트/매트릭스 다루기

백트래킹/분지한정(작은 N), 혹은 휴리스틱(DSATUR/Greedy) 설계하기

최적화 문제의 정확해법 vs 근사해법 트레이드오프 이해

문제 설명
정점 수 NNN과 간선 수 MMM인 무방향 단순 그래프가 주어진다. 인접한 두 정점은 서로 다른 색으로 칠해야 한다. 이때 **필요한 최소 색의 수(Chromatic Number)**를 출력하라.

정의: Chromatic Number χ(G)\chi(G)χ(G)는 그래프 GGG의 정점들을 인접 제약을 만족하도록 색칠할 때 필요한 최소 색의 수이다

입력 형식
첫째 줄: 정점 수 NNN (1 ≤ N ≤ 1000), 간선 수 MMM (0 ≤ M ≤ 50,000)

다음 MMM줄: 간선 u vu\ vu v (0 ≤ u, v < N, u ≠ v)

중복 간선 없음, 자기 루프 없음

정점은 0부터 N−1N-1N−1까지 번호

출력 형식
한 줄에 최소 색 수 χ(G)\chi(G)χ(G) 출력

제한/채점 규칙 (2단계)
Subtask 1 (40점): N≤30N \le 30N≤30. 정확한 최솟값을 출력해야 한다. (백트래킹/분지한정, 비트마스크 가능)

Subtask 2 (60점): N≤1000N \le 1000N≤1000, M≤50,000M \le 50{,}000M≤50,000. 정확한 값이 요구된다.

권장 전략: 상계/하계 결합 + DSATUR 기반 분지한정(Pruning 강함).

단, 시간 초과를 방지하기 위해 강한 정렬/가지치기를 구현하라.

대체 허용(부분점수 없음): 단순 Greedy/Welsh-Powell만으로는 최악 입력에서 오답 가능.
